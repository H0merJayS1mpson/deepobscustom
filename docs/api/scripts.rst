============
Scripts
============

DeepOBS includes a few convenience scripts that can be run directly from the
command line

  - **Prepare Data**: Takes care of downloading and preprocessing all data sets
    for DeepOBS.
  - **Get Baselines**: Automatically downloads the baselines of DeepOBS.
  - **Plot Results**: Quickly plots the suggested outputs of a optimizer benchmark.
